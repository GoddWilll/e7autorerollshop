# e7autorerollshop
A script made in order to automatically reroll the secret shop.
It is still in development.

# Requirements
- python3
- Pillow
- OpenCV
- pyautogui
- keyboard

# How to install
```
pip install Pillow

pip install opencv-python

pip install pyautogui

pip install keyboard
```
